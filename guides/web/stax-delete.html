stax delete

Clean up your local branches that no longer exist on the remote

About This Command

The stax delete command helps you clean up your repository by removing local branches that have been deleted from the remote (often called "gone" branches). This keeps your branch list tidy and prevents confusion when working with your repository.

Interactive Terminal Demo

How To Use

The command scans your local repository for branches that are tracking remote branches that no longer exist and removes them.

Basic Usage:

stax delete

What It Does:

  1. Fetches latest information from the remote repository
  2. Identifies local branches that are tracking deleted remote branches
  3. Removes these "gone" branches from your local repository
  4. Provides a summary of removed branches

Benefits

  • Keeps your repository clean and organized
  • Reduces confusion when switching between branches
  • Improves Git command output by removing references to non-existent branches
  • Helps maintain better repository hygiene
