stax delete

Efficiently remove gone branches from your Git repository. This command helps clean up branches that no longer exist on the remote, keeping your repository organized.

Key Features:

  • Identifies and removes branches that have been deleted from the remote
  • Helps maintain a clean local repository
  • Safe operation with clear feedback
